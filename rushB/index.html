RushB.js 中文文档

Intro 概述

RushB.js是一个没有经过太多优化的js动画库，能够实现包括颜色动画，转换(transform)动画在内的缓动动画；在处理transfrom，颜色的方法上参考了一点velocity.js的早期版本，但是主体上还是从零实现，和几千行的velocity.js相比想必有很大差距 (´c_`)。

Basic Usage 基础用法

下面是一个简单的例子：


        <body>
            ...
            <script src="rushB.js"></script>
        </body>
                        

    // 创建rush对象
    var rush1 = new Rush(block1);

    // add方法添加动画队列
    rush1.add({
        width: 300, // 单位可省略
    }, 1500);

    rush1.add({
         'width': '70px',
         'margin-left': 200 // 连字符形式
    }, 2000);

    // 启动动画
    rush1.start();

由于RushB.js支持链式调用，所以上面的代码等同于：


    // 支持链式调用
    var rush1 = new Rush(block1).add({
        width: 300,
    }, 1500).add({
        'width': 70,
        'margin-left': 200
    }, 2000).start();
开始

Transform 动画

transform动画有一点尴尬，与velocity.js一样，无法获取到元素的初始transform属性，所以仅适用于初始状态没有transform属性的元素，transform问题已解决，详见俺的水文；且要注意的是，定义transform属性时需要以相关属性而非transform为属性名，以下是所有常用的transform可用属性： 


    {
        /* translate */
        'translateX': 20,     // 等同于'20px'
        'translateY': '1.5em',
        'translateZ': '20px', // IE10+
        /* scale */
        'scale': 0.5,
        'scaleX': 0.5,
        'scaleY': 0.5,
        /* rotate */
        'rotateX': '45deg',
        'rotateY': '45deg',
        'rotateZ': '45deg',
        /* skew */
        'skewX': '30deg',
        'skewY': '30deg'
    }

一个应用transform动画的例子：


    var rush2 = new Rush(block2).add({
        translateX: 200,    // 注意定义，不是transform: translateX(200px)
        rotateZ: 360        // deg可省略
    }, 1000).add({
        translateX: 0,
        rotateZ: '0deg'
    }, 1000);

    rush2.start();
开始

Color 动画

color动画支持十六进制，hsl，rgb，rgba的颜色表示，所有支持的颜色属性如下所示： 


    {
        'color': '#fff',
        
        'background-color': '#fff',

        'border-color': '#fff',
        
        'outline-color': '#fff'
    }

一个应用color动画的例子：


    var rush3 = new Rush(block3).add({
        'background-color': '#C54CFF'
    }, 1000).add({
        'background-color': 'rgba(79, 249, 255, 0.8)'
    }, 1000).start();
开始

Arguments 参数

add方法用于添加动画序列，下面为参数的写法： 


    {
        rush.add(
        // 第一个参数，元素末状态属性
        {
            // 动画属性，宽度到'500px'的动画                             
            width: '500px',
            // 属性示例
            property2: value2 
        },
        // 第二个参数，动画时长，单位毫秒
        1000,
        // 第三个参数，动画额外配置项，可不传
        {                                    
            // 此动画任务开始前触发的函数
            // 如设置循环，只在第一次触发
            // 如果有delay，在delay结束后才触发              
            before: function() {},
            // 此动画任务结束后立即触发的函数
            // 如设置循环，只在最后一次触发
            after: function() {},
            // 此动画任务开始前的延时，单位毫秒
            delay: 1000,
            // 缓动函数，具体参照Tween.js文档
            easing: Math.tween.Quad.easeInOut,
            // task结束后设置元素的display
            display: 'block'
        });
    }

除此之外，也提供了几个预定义的常用动画指令，将字符串作为第一个参数传入add方法即可，目前已支持的有"slideUp" && "slideDown" 和 "fadeOut" && "fadeIn"，下面是一个例子：


    var rush4 = new Rush(block4);
    rush4.add('slideUp', 1000);
    rush4.add('slideDown', 1000, {
        after: function() {
            alert('p90 rush b, dont stop!');
        },
        delay: 600 // 600ms的延迟
    });

    rush4.start();
开始

Command 指令

为了实现对动画的控制，俺们提供了一些基本的指令，基本到用一个例子就能介绍完 (´･_･`)： 


    var rush5 = new Rush(block5);
    rush5.add({
        translateX: 300,
        rotateZ: 360
    }, 800);
    rush5.add({
        translateX: 0,
        rotateZ: 0
    }, 1200, {
        after: function() {
            alert('p90 rush b dont stop!')
        }
    });

    // setLoop方法，传入动画重复的次数，如果不传入参数则无限重复
    // 设置了循环后，如有before方法则只在第一轮动画触发
    // 如有after方法则只在最后一轮动画触发（如果是无限循环则永远不会触发）
    // rush5.setLoop(); 将无限重复动画
    // rush5.setLoopForever(); 作用与上相同，更加语义化
    rush5.setLoop(3); 

    btn5_1.onclick = function() {
        rushHandler[rush5.state](rush5, btn5_1);
    };

    btn5_2.onclick = function() {
        btn5_1.innerHTML = '开始';

        // finish方法，将直接结束已启动的动画(无过渡)
        // 即使是无限循环也将直接结束，且不触发after回调
        rush5.finish(); 
    }

    var rushHandler = {
        0: function(rush, btn) { // STATE_INITIAL 动画未启动
            btn.innerHTML = '暂停';

            // start方法，启动动画序列
            rush.start(); 
        },
        1: function(rush, btn) { // STATE_START 动画正在进行
            btn.innerHTML = '继续';

            // pause方法，暂停已启动的动画
            rush.pause();
        },
        2: function(rush, btn) { // STATE_STOP 动画暂停中
            btn.innerHTML = '暂停';

            // play方法，继续已暂停的动画
            rush.play();
        }
    }
开始 结束
