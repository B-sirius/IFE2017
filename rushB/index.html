RushB.js

概述

RushB.js是一个没有经过太多优化的js动画库，能够实现包括颜色动画，转换(transform)动画在内的缓动动画，在处理transfrom，颜色的方法上参考了一点velocity.js的早期版本，但是主体上还是自己实现，和几千行的velocity.js想必有很大差距（摊）。

如何使用

由于希望实现尽可能多的缓动效果，RushB.js依赖Tween.js。下面是一个简单的例子：


                            'use strict';

                            var a = function() {
