RushB.js 中文文档

Intro 概述

RushB.js是一个没有经过太多优化的js动画库，能够实现包括颜色动画，转换(transform)动画在内的缓动动画，在处理transfrom，颜色的方法上参考了一点velocity.js的早期版本，但是主体上还是从零实现，和几千行的velocity.js相比想必有很大差距（摊）。

Basic Usage 基础用法

由于希望实现尽可能多的缓动效果，RushB.js依赖Tween.js。下面是一个简单的例子：


        <body>
            ...
            <script src="tween.js"></script>
            <script src="rushB.js"></script>
        </body>
                        

        // 创建rush对象
        var rush1 = new Rush(block1);

        // add方法添加动画队列
        rush1.add({
            width: 300,
            height: '200px'
        }, 1500);

        rush1.add({
             'width': 70,
             'height': 70,
             'margin-left': 200 // 连字符形式
        }, 2000);

        // 启动动画
        rush1.start();

由于RushB.js支持链式调用，所以上面的代码等同于：


        // 支持链式调用
        var rush1 = new Rush(block1).add({
            width: 300,
            height: '200px'
        }, 1500).add({
            'width': 70,
            'height': 70,
            'margin-left': 200
        }, 2000).start();
开始

Transform 动画

transform动画有一点尴尬，与velocity.js一样，无法获取到元素的初始transform属性，所以仅适用于初始状态没有transform属性的元素，且要注意的是，定义transform属性时需要以相关属性而非transform为属性名，以下是所有常用的transform可用属性： 


        {
            /* translate */
            translateX: 20,     // 等同于"20px"
            translateY: "1.5em",
            translateZ: "20px", // IE10+

            /* scale */
            scale: 0.5,
            scaleX: 0.5,
            scaleY: 0.5,

            /* rotate */
            rotateX: "45deg",
            rotateY: "45deg",
            rotateZ: "45deg",

            /* skew */
            skewX: "30deg",
            skewY: "30deg",
        }

一个应用transform动画的例子：


        var rush2 = new Rush(block2).add({
            translateX: 200,    // 注意定义，不是transform: translateX(200px)
            rotateZ: 360        // deg可省略
        }, 1000).add({
            translateX: 0,
            rotateZ: '0deg'
        }, 1000);

        rush2.start();
开始

Color 动画

color动画支持十六进制，hsl，rgb，rgba的颜色表示，所有支持的颜色属性如下所示 


        {
            'color': '#fff',
            
            'background-color': '#fff',

            'border-color': '#fff',
            
            'outline-color': '#fff'
        }

一个应用color动画的例子：


        var rush3 = new Rush(block3).add({
            'background-color': '#C54CFF'
        }, 1000).add({
            'background-color': 'rgba(79, 249, 255, 0.8)'
        }, 1000).start();

Arguments 参数

add方法用于添加动画序列，下面为参数的写法： 


        {
            rush.add({
                // 第一个参数，需要变换的属性对象                                  
                width: '500px',                         // 动画属性，宽度到'500px'的动画
                property2: value2                       // 属性示例
            },1000, { // 第二个参数，动画时长，单位毫秒                                   
                // 第三个参数，动画额外配置项，可不传                                        
                before: function() {},                  // 此动画任务开始前触发的函数（如果有delay，在delay结束后才触发）
                after: function() {},                   // 此动画任务结束后立即触发的函数
                delay: 1000,                            // 此动画任务开始前的延时，单位毫秒
                easing: Math.tween.Quad.easeInOut,      // 缓动函数
                display: 'block'                        // task结束后设置元素的display
            });
        }

除此之外，也提供了几个预定义的常用动画指令，将字符串作为第一个参数传入add方法即可，目前已支持的有"slideUp" && "slideDown" 和 "fadeOut" && "fadeIn"，下面是一个例子：


    var rush4 = new Rush(block4);
    rush4.add('slideUp', 1000);
    rush4.add('slideDown', 1000, {
        after: function() {
            alert('p90 rush b, dont stop!');
        },
        delay: 600 // 600ms的延迟
    });

    rush4.start();
开始
